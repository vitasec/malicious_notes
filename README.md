Malicious Notes 🛠️

A collection of scripts and tools designed for security research, penetration testing, and educational purposes. This repository contains various utility scripts including reverse shells and credential processing tools.
📂 Repository Contents
File	Description
conv.py	A Python script that processes password lists. It cleans special characters, generates MD5 hashes, prepends "admin:", and outputs the final result in Base64 encoding.
phpshell.php	Devidoff's Reverse Shell Paradise. A sophisticated PHP web shell with an authentication system. Features include command execution, system information gathering, and a multi-language reverse shell generator (Bash, PHP, Perl, Python, Ruby, etc.).
shell.sh	A classic Bash reverse shell one-liner directed to a predefined internal IP and port.
